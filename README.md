# PaperSearch — 學術文獻搜尋工具

Search academic literature filtered by Impact Factor using OpenAlex API.

- Search by topic keywords
- Filter by minimum Impact Factor (2yr Mean Citedness)
- Filter by publication year range
- Export results as TXT, PDF, or Word (DOCX)
